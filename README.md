HSV FInder 
====================== 
This is a Kinect project I built to find the HSV (Hue, Saturation, Value) range for objects under specific lighting. 

How To Use 
======================= 
Simply run this app on Kinect (first generation) using Visual Studio 2012 (or later) IDE.
